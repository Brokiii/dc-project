#Dokumenty Cyfrowe - projekt

### How to run?

1. run ```docker build -t my-postgres-db ./``` in terminal to build docker database container
2. run ```docker run -p 5432:5432 my-postgres-db``` to run that container
    - port: 5432
    - db name: insurance
    - username: insurance
    - password: pass
